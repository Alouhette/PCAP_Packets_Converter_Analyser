Project from the Winter 2026 Semester at the UQAC (Change can happen in the futur, this is only a V1)



What does this project do :
    - Transform a .pcap file into a CSV file 
    - Think about potential ways to identify suspicious packets
    - Exploit the data in the CSV file to identify the potential suspicous packets

(Please consider this project is still under progress and that errors & changes may appear)


To see how to use each functions please refer to Main.java
You will be able to test out the outputs with 2 given .pcap files that ALREADY are converted to CSV
